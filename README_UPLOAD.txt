GitHub Pages Upload Folder

This folder contains the static website files ready for GitHub Pages.

How to use:
1. Open your GitHub repo: https://github.com/marleypumpkin08-ctrl/phantom-fire-games-site
2. Upload all files from this folder to the repo root (or replace existing files).
3. Commit changes.
4. In repo Settings > Pages, set Source to GitHub Actions (if workflow exists) or Deploy from branch (main / root).

Website files are:
- index.html
- assets/
- .nojekyll
